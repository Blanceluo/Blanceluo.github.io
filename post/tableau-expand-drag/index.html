介绍

目前本人所在的是一家主营广告营销平台的公司，主要是利用公司在海外广告浸淫多年的行业经验，帮助国内各大中小型企业出海的广告投放。 其中也不乏快手，抖音这样的人人皆知的客户。

目前技术部门主要是为公司工具化的开发。因某些因素的缘故，公司对技术部门的业务战略进行了重新的定位，以及期望的提高。 致本人目前负责的若干个项目可能有 break change级的变化。 最大的变化是可能会引入计算引擎，以此帮助我们对在项目中的数据有了更高级的分析能力。 Tableau 便是BI范畴内的一个数据可视化分析服务，不同于我们对于简单的数据可视化框架的认知。 它集成的是一整套服务, Desktop，Server，Online 等等。

让我对它们有兴趣的不是这所谓的优质服务，而是一些操作交互方式。页面的模块可拖曳。 实现类似的功能很简单，但若要做好整个页面的模块交互可非我们所想的那么简单。

(() => {
  let startY = 0
  let endY = 0
  let nodeLine = null

  const moveHandler = e => {
    endY = e.clientY
    nodeLine.style.top = endY - startY + 'px'
  }

  const downHandler = e => {
    e.preventDefault()

    if (e.target.dataset['type']) {
      nodeLine = e.target
      startY = e.clientY

      document.addEventListener('mousemove', moveHandler, false)
      document.addEventListener('mouseup', upHandler, false)
    }
  }

  const upHandler = e => {
    const parent = nodeLine.parentNode
    const height = +(window.getComputedStyle(parent, null).getPropertyValue('height').replace(/px/g, ''))
    const marginTop = +(window.getComputedStyle(parent, null).getPropertyValue('margin-top').replace(/px/g, ''))

    parent.style.cssText = `
      height: ${height - (endY - startY)}px;
      margin-top: ${marginTop + (endY - startY)}px;
    `
    nodeLine.style.top = '0'

    // 状态重置
    startY = 0
    endY = 0
    nodeLine = null

    // 移除绑定事件
    document.removeEventListener('mousemove', moveHandler, false)
    document.removeEventListener('mouseup', upHandler, false)
  }

  document.addEventListener('mousedown', downHandler, false)
})()
